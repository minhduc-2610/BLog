Tuỳ Chỉnh CSS Cho Hình Ảnh

Bạn có thể thêm CSS để tùy chỉnh giao diện hiển thị của hình ảnh.

.img-responsive {
max-width: 100%;
height: auto;
display: block;
margin: 0 auto;
}

Chú Ý

Đảm bảo rằng hình ảnh được lưu trong thư mục static/ để có thể truy cập trực tiếp thông qua đường dẫn.

Ví dụ, hình ảnh đặt tại static/images/example.jpg sẽ được truy cập qua /images/example.jpg.

Với các bước trên, bạn có thể dễ dàng thêm và tùy chỉnh hình ảnh trong Hugo!
